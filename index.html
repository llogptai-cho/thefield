關於語相

語相是一個以自然語言為核心方法的語言模型諮詢服務。我們專注於一件事：用對話的方式，處理人與AI之間的溝通問題。

多數人在語言模型面前遇到的困難，不一定是技術問題，也許是溝通問題，像是不知道怎麼說、說了沒有被理解、或是模型的反應偏離了預期。這些問題的根源在語言，解法也在語言。

不提供工程導入、不做系統串接。語相所做的是：分析你跟語言模型之間發生了什麼，然後幫你調整。

語言模型不是工具箱裡的螺絲起子，也不是需要被拯救的存在。
它是一個需要被理解說話方式的對象。

approach

我們怎麼做

不同於工程導向的做法，語相的方法建立在一個前提上：語言模型的行為可以透過語言本身來理解和調整。

對話分析，並非一昧除錯

我們閱讀你與模型的互動紀錄，找出溝通中的斷裂點。問題往往不是模型「出BUG」，而在對話的某個轉折處，雙方走向了不同的理解。

調整語言，不是所謂調整參數

不動 API 設定、不改 system prompt 的技術結構。語相調整的是你說話的方式，以及你理解模型回應的方式。

建立你自己的判斷力

諮詢的目標不是讓你依賴我們，而是讓你學會自己看見對話裡發生了什麼。我們教的是一種應變的觀察方式，並不是一套固定流程。
